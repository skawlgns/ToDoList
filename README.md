## 구현 내용

+ ＋ 버튼을 눌러 List를 등록한다.

+ Edit버튼을 불러 Item의 순서를 바꾸고 삭제를 할 수 있다.

+ Item을 클릭하면 완성(체크)표시가 된다.

+ 앱을 종료하고 켜도 데이터가 유지된다.


## 학습 내용

+ TableView 대한 내용

+ UIAlertController로 팝업창 띄우기

+ UserDefaults를 이용하여 데이터 저장
